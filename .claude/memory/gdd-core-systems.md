# GDD 核心系统规则（v3.3 摘录）

## 时间系统
- 1游戏日 = ~14分钟现实时间
- 6:00 AM 起床 → 凌晨 2:00 AM 强制昏迷
- 1季 = 28天，1年 = 112天
- 流速：1游戏分钟 = 0.7秒现实时间
- 昏迷惩罚：丢失10%金钱（最多5000g）
- 联机：主机控制时间流速

## 经济系统
- 唯一货币：金币（g），初始500g
- 品质倍率：normal×1.0 / silver×1.25 / gold×1.50 / iridium×2.0
- 工匠专精加成：×1.4
- 经济阶段：
  - 初期（春1-14）：日均200-500g
  - 发展期（春14-夏）：日均1,000-5,000g
  - 成长期（第1年秋）：日均5,000-15,000g
  - 成熟期（第2年）：日均15,000-50,000g
  - 自由期（第3年+）：日均50,000g+
- 安全上限：金钱 999,999,999 / 单物品价格 100,000 / 日收入 500,000

## 体力系统
- 初始最大体力：270
- 每次升级 +5（上限999）
- 工具消耗：-2~-8/次（不随等级变化）
- 归零：不能用工具，可走路/对话
- 恢复：睡觉全恢复 / 吃食物部分恢复 / 温泉缓慢恢复

## 天气系统（季节概率矩阵）
| 季节 | 晴 | 雨 | 雷暴 | 雪 | 大风 |
|:----:|:--:|:--:|:----:|:--:|:----:|
| 春 | 58% | 25% | — | — | 17% |
| 夏 | 55% | 25% | 12% | — | 8% |
| 秋 | 50% | 25% | 5% | — | 20% |
| 冬 | 45% | — | — | 45% | 10% |

天气核心影响：
- 雨天：自动浇水、NPC外出减少、钓鱼咬钩×1.3
- 雷暴：可能劈中设施（需避雷针保护）
- 雪：不能种地、冬季采集物出现
- 大风：作物倒伏+15%

## 联机系统核心规则
- 架构：Listen Server（主机兼玩家）
- 最大4人（代码层硬上限8人）
- 局域网自动发现，互联网房间码+NAT穿透
- 客户端预测+主机仲裁：操作0感知延迟
- 金钱模式：共享/独立（创建时选定不可改）
- 主机掉线：保留5分钟重连窗口
- 网络限流：60消息/秒、50KB/秒上行、16KB单消息上限

## 存档系统
- 3手动槽位 + 1自动存档
- JSON + sha256 校验
- 原子写入（先临时文件后改名）
- 覆盖前先备份
- 版本迁移：逐版本应用迁移函数

## 系统整合闭环（6条反馈链）
1. 耕种 → 烹饪 → 战斗/采矿 → 升级工具 → 更高效耕种
2. 动物 → 工匠 → 经济 → 建筑 → 更多动物
3. 采矿 → 合成 → 洒水器/避雷针 → 提高耕种效率
4. 战斗 → 怪物材料 → 合成设施 → 被动产出
5. 钓鱼 → 烹饪 → NPC送礼 → 好感度 → 解锁配方
6. NPC好感度 → 赠送配方/帮忙 → 效率提升 → 更多社交
