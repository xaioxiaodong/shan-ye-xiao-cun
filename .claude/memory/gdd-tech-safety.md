# GDD 技术与安全规范（v3.3 摘录）

## 代码规范（不可变更）
- 全栈 TypeScript，禁止 any（用 unknown 替代）
- 所有变量/参数/返回值显式类型声明
- 命名：camelCase（变量/函数），PascalCase（类/接口/类型）
- 文件命名：kebab-case.ts
- 单文件 ≤ 300 行，单函数 ≤ 50 行
- 注释：中文 JSDoc

## 安全防护（7 维熔断保护）

### 11.4.1 游戏循环安全
- 帧时间上限：100ms/帧，超限跳过剩余
- 迭代上限：10,000 次/update
- 看门狗：5秒检查，3秒阻塞→重启场景

### 11.4.2 渲染安全
- 精灵上限：全局 5,000 / 每层 1,000
- 粒子上限：每发射器 200 / 同时 20 个发射器
- 纹理内存上限：256MB
- Tile 裁剪：视口外 +2 tile 缓冲，单帧 ≤ 10,000 tile

### 11.4.3 网络安全
- 速率限制：60 消息/秒，50KB/秒上行
- 单消息上限：16KB
- 连接超时：初次 10s，空闲 5min 断开
- 状态验证：位置合法性、最大移速 8 tile/s、金钱变动 ≤ 50,000/min

### 11.4.4 内存安全
- 场景切换：清理纹理缓存/音效/Tween/GC
- 资源加载超时：15s/资源
- 缓存上限：纹理 128MB / 音频 64MB / 数据 500 条
- 对象池：最多 50 池 / 每池 500 对象

### 11.4.5 数据安全
- 原子写入 + sha256 校验 + 覆盖前备份
- 数值边界：金钱 0-999,999,999 / 体力 0-999 / HP 0-999 / 好感 0-2,500 / 堆叠 0-999 / 技能 0-10
- 恢复：自动存档恢复 / 3 个备份槽 / 损坏时提示用户

### 11.4.6 状态机安全
- 非法状态转换：回退并记录
- NPC 日程异常：使用默认位置（小镇中心）
- 任务一致性：读档检查 + 自动修复
- 物品 ID：白名单验证，拒绝未知 ID

### 11.4.7 文件 I/O 安全
- 单文件 ≤ 10MB
- 允许扩展名：.json .png .wav .ogg .mp3 .tsx .tmx
- 设置文件：验证→损坏用默认→自动恢复

### 11.4.8 联机安全
- 玩家硬上限 8（设计 4）
- 主机保护：4 客户端 / 50KB/客户端 / 过载降频→踢人
- 消息队列：最多 1,000 条 / 每帧处理 10ms
- 作弊防护：操作 ≤ 20 次/秒 / 50 次标记可疑

## 网络协议消息类型（14种）
```
玩家操作：p:move / p:action / p:talk / p:chat
状态同步：s:player / s:farm / s:npc / s:time
交互：i:trade / i:gift / i:help
系统：sys:save / sys:error / sys:kick / sys:host
```

## 错误恢复策略（7类）
| 异常 | 检测 | 恢复 | 回退 |
|------|------|------|------|
| 存档损坏 | checksum/json/字段缺失 | 恢复备份/新建/提示 | 用自动存档 |
| 网络断开 | 超时/心跳/拒绝 | 自动重连/5秒重试 | 离线模式 |
| 资源加载失败 | 超时/404/解码错误 | 跳过/重试/占位纹理 | 缺失纹理占位 |
| 渲染崩溃 | WebGL丢失/内存溢出 | 重启渲染/降质/重载 | 错误提示退出 |
| 任务状态异常 | 目标数/前置/标记 | 自动修复/重置/标记 | 记录并跳过 |
| 玩家位置异常 | 越界/碰撞/地下 | 传送出生点/上一安全点 | 回农场 |
| 时间异常 | 倒退/跳跃/跳日 | 回退有效值/截断 | 强制睡眠存档 |

## 开发风险清单
| 风险 | 可能性 | 影响 | 缓解 |
|------|:------:|:----:|------|
| 联机延迟过高 | 中 | 高 | 客户端预测+插值 |
| 存档损坏丢档 | 低 | 极高 | 原子写入+校验+备份 |
| 大量作物降帧 | 中 | 中 | 渲染裁剪+精灵上限 |
| 网络消息洪水 | 低 | 高 | 速率限制+队列保护 |
| 数据溢出 | 低 | 中 | 数值边界保护 |
| NPC日程异常 | 中 | 中 | 默认位置回退 |
| 手机内存不足 | 中 | 高 | 缓存上限+场景清理 |
| 多人抢地冲突 | 低 | 中 | 主机仲裁+客户端回滚 |
