# 第一部分：游戏总纲

## 1.1 游戏名称

**暂定名：** 《山野小村》（暂定，可更换）

## 1.2 一句话定义

> 一款像素风格的乡村生活模拟游戏，玩家继承一座废弃农场，在田园小镇中种田、养殖、交友、探索，重建生活。

## 1.3 设计哲学

本游戏的设计围绕三条核心理念展开，所有设计决策必须与之对齐：

| 理念 | 含义 | 设计影响 |
|------|------|----------|
| **舒适循环** | 玩家的每一天都应该是"有收获、有期待、不焦虑"的 | 无强制失败、无时间限制、无惩罚性随机（→ 14.1） |
| **内容密度 > 系统广度** | 宁可深耕少数系统，也不做一堆浅尝辄止的功能 | 每个 P0 系统必须有足够的内容支撑（→ 4.1–4.14） |
| **有机整合** | 每个系统不是孤立的，它的存在必须滋养另一个系统 | 系统整合闭环检查（→ 5.3） |

## 1.4 核心设计原则（不可违背）

| # | 原则 | 说明 |
|---|------|------|
| 1 | **体验优先，数值次之** | 任何数值系统必须服务于"舒适放松"的核心体验（→ 3.2 经济曲线验证） |
| 2 | **内容密度决定质量** | 宁可系统少，不可内容浅。每个系统必须有足够的内容支撑 |
| 3 | **渐进解锁，不提前给完** | 所有内容分阶段释放，玩家始终有"下一个目标"（→ 5.4 进度门控表） |
| 4 | **没有强制失败** | 不设 Game Over，没有时间限制，玩家永远可以按照自己的节奏玩 |
| 5 | **像素美术风格统一** | 16×16 Tile 基准，有限色板，手绘风格 |
| 6 | **全平台可玩** | PC（Tauri）和手机（Capacitor）共用一套代码 |
| 7 | **质量与防护并重** | 合理优化是必要的。**必须设置安全防护措施**防止 bug 或异常导致过载/崩溃（→ 11.4 七维熔断保护） |
| 8 | **联机不歧视** | 所有玩家体验完全一致，不存在主机优势（→ 8.4 客户端预测+主机仲裁） |
| 9 | **系统整合闭环** | 所有核心系统必须至少与另外两个系统有正向反馈循环（→ 5.3 整合检查清单） |
| 10 | **可及性优先** | 支持色盲模式、可重映射键位、触屏与键鼠同样完整（→ 12.1） |

## 1.5 目标玩家画像

| 维度 | 画像 |
|------|------|
| 年龄 | 16-35 岁 |
| 游戏经验 | 轻度到中度玩家 |
| 核心诉求 | 放松、有成就感、有陪伴感 |
| 平台偏好 | PC 为主，手机碎片时间 |
| 同类游戏经验 | 玩过或听说过星露谷物语 |
| 单次游戏时长 | PC 20-60 分钟（1-4 天游戏内）；手机 5-15 分钟（1 天游戏内） |
| 付费意愿 | 买断制 20-40 元区间，接受 DLC 扩展 |
| 社交需求 | 喜欢分享但不喜欢强制社交 |

# 第二部分：世界观与设定

## 2.1 故事背景

> 你在大城市做着一份不痛不痒的工作，每天挤地铁、吃外卖、加班到深夜。  
> 某天，你收到一封来自远房亲戚的信：  
> "我年纪大了，山脚下的老房子和地没人管。你要是愿意来，就归你了。"  
> 你辞了工作，坐上长途车，来到了这个叫 **青溪镇** 的地方……

```
温暖、治愈、慢生活
```

### 售价计算规则

```typescript
// 售价计算公式（全局统一）
// 安全保护：输出值受 11.4.5 valueBounds.money 保护，防 NaN/Infinity
function calculateSellPrice(
  basePrice: number,
  quality: 'normal' | 'silver' | 'gold' | 'iridium',
  isArtisan: boolean,
  hasPerk: boolean,
): number {
  const qualityMultiplier: Record<string, number> = {
    normal: 1.0, silver: 1.25, gold: 1.50, iridium: 2.0,
  };
  const multiplier: number = qualityMultiplier[quality] ?? 1.0;
  const artisanMultiplier: number = isArtisan && hasPerk ? 1.4 : 1.0;
  const result: number = basePrice * multiplier * artisanMultiplier;
  if (!isFinite(result)) return 0;
  return Math.floor(Math.max(0, result));
}
```

### 3.2.1 经济计算实例

以下为实际游戏中的经济链示例，展示各系统的经济价值：

```
【初级经济链：直接出售】
防风草种子 20g → 种植4天 → 防风草售价 35g → 利润 15g（日均 3.75g）

【中级经济链：加工增值】
番茄种子 30g → 种植11天 → 番茄售价 60g → 腌菜桶25天 → 腌番茄 120g
  → 利润 90g（日均 3.75g → 加工后日均等效 4.8g）

【高级经济链：工匠+专精】
蓝莓种子 50g → 种植13天（再生4天）→ 一季收获4次（第13/17/21/25天）
  → 每次80g × 酿酒桶6天 × 3.0倍率 = 240g/批
  → 4批全完成（最后一批跨季出桶，不影响）= 960g
  → 工匠专精×1.4 → 1,344g
  → 总利润 1,344 - 50 = 1,294g（按季节28天计，日均 46.2g）

【动物经济链】
鸡 800g + 鸡舍 3,000g = 3,800g 初始投入
  → 每天产蛋 1个（蛋基础价 100g）→ 蛋黄酱机3h → 蛋黄酱 180g（×1.8）
  → 日收入 180g → 回本周期 22天

【终局经济链：大规模自动化】
铱洒水器覆盖 5×5 → 25格自动浇水
  → 每格种上古水果（再生7天）→ 酿酒桶 → 果酒
  → 每格日收入约 150g → 25格日收入 3,750g
  → 被动收入（只需收获+放桶）
```

### 4.14.2 节日详细玩法

**春之祭——种植比赛**
```
1. 玩家在春14前通知时报名参赛
2. 提交一个作物参赛（选择品质最高的）
3. NPC评委根据品质/稀有度评分
4. 名次奖励：第一名 3,000g + 肥料大礼包
              第二名 2,000g + 高级肥料×5
              第三名 1,000g + 基础肥料×5
              参与奖   500g
5. 联机模式：所有玩家可分别参赛，互不影响
```

**夏夜花火——烟花大会**
```
1. 18:00 在小镇广场集合，自动触发活动
2. 20:00 烟花开始（播放烟花动画）
3. 烟花期间，与NPC对话好感度+50（日常+20）
4. 可与特定NPC一起看烟花（选择后触发特殊对话）
5. 结束后获得限定道具"烟花棒"
6. 联机模式：所有玩家一起看烟花，可互相赠送烟花棒
```

**丰收节——农产品展评**
```
1. 玩家提交工匠产品参展（蛋黄酱/奶酪/果酒等）
2. 评分标准：售价×品质系数
3. 排名奖励：第一名 5,000g + 酿酒桶×1
              第二名 3,000g + 高级保湿土×5
              第三名 2,000g + 基础肥料×10
4. 所有参展产品不消耗（展示后返还）
5. 联机模式：可一起参加，评比各自独立
```

**冬雪庆典——烹饪比赛**
```
1. 提交料理参赛
2. 评分标准：售价×buff效果
3. 排名奖励：第一名 3,000g + 稀有食谱×1
              第二名 2,000g + 高级食材×5
              第三名 1,000g + 普通食谱×1
4. 赛后获得限定食谱"冬雪甜汤"
5. 联机模式：可一起参加，评比各自独立
```

## 5.2 系统间正向反馈链

### 5.2.1 耕种 → 烹饪 → 战斗/采矿
```
耕种产出作物 → 烹饪成料理 → 食用获得属性增益 → 提高战斗/采矿效率
                                                      ▼
                                        获得更高级矿石/掉落 → 升级工具 → 更高效耕种
```

### 5.2.2 动物 → 工匠 → 经济 → 建筑
```
动物产原料 → 工匠设备加工 → 高价出售 → 获得金币 → 建造更多建筑 → 养更多动物
```

### 5.2.3 采矿 → 合成 → 耕种/养殖
```
采矿获得矿石 → 合成洒水器/避雷针/种子机 → 提高耕种效率 → 释放时间做其他事
```

### 5.2.4 战斗 → 合成
```
战斗掉落怪物材料 → 合成特殊设施（避雷针/宝石复制机） → 被动产出资源
```

### 5.2.5 钓鱼 → 烹饪 → NPC送礼
```
钓鱼获得鱼类 → 烹饪成料理 → 赠送给NPC → 好感度提升 → 解锁配方/剧情/帮助
```

### 5.2.6 NPC → 玩家 → 正向循环
```
NPC好感度提升 → 赠送配方/材料/帮忙干活 → 玩家效率提升 → 更多时间社交 → 更高好感度
```

```typescript
class ClientPrediction {
  performAction(action: PlayerAction): void {
    this.localState.apply(action);
    this.network.sendAction(action, (serverResult: ActionResult) => {
      if (!serverResult.success) {
        this.localState.rollback(serverResult.correctState);
        this.showFeedback('conflict', serverResult.message);
      }
    });
    this.localState.confirmOptimistic(action);
  }

  interpolateOtherPlayer(playerId: string, renderTimestamp: number): void {
    const player = this.network.getPlayerState(playerId);
    const t = (renderTimestamp - player.lastUpdate) / player.updateInterval;
    this.sprite.position = Phaser.Math.Linear(player.prevPos, player.targetPos, t);
  }
}
```
